Estudo dos algoritmos dos métodos de Euler e Runge-Kutta de 4ª ordem

O trabalho tem como objetivo desenvolver uma aplicação em linguagem Java que modela o consumo epidêmico de drogas utilizando um sistema de equações diferenciais ordinárias (EDOs) de primeira ordem. Foi utilizado o modelo SIR (Suscetíveis, Infetados, Recuperados) para representar a dinâmica do consumo de drogas na população. Recorreu-se ao uso dos métodos numéricos de Euler e Runge-Kutta de 4ª ordem para resolver as EDOs e no final efetuou-se a comparação dos resultados de ambos os métodos.
